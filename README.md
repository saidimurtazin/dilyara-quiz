# 🎂 Викторина «Насколько хорошо ты знаешь Диляру?»

Готовый проект: страница с викториной (25 вопросов), ввод имени, общая таблица
лидеров и статистика. Нужно только подключить базу и задеплоить — получится
публичная ссылка, которую можно разослать гостям.

---

## 📁 Что внутри

```
dilyara-quiz/
├── public/
│   ├── index.html        ← вся викторина (фронтенд)
│   └── photos/           ← СЮДА класть фото Диляры (создать папку)
├── api/
│   └── submit.js         ← сохраняет результаты и отдаёт топ-5
├── supabase_schema.sql   ← SQL для создания таблицы в базе
├── package.json
└── README.md (этот файл)
```

## 🚀 Инструкция для Claude Code

Скажи Claude Code: **«Разверни этот проект по шагам из README»**. Шаги:

### Шаг 1. База данных (Supabase, бесплатно)
1. Зарегистрируйся на https://supabase.com → New project (запомни пароль).
2. Открой **SQL Editor → New query**, вставь содержимое `supabase_schema.sql`, нажми **Run**.
3. Зайди в **Project Settings → API** и скопируй два значения:
   - `Project URL`  → это `SUPABASE_URL`
   - `anon public`  ключ → это `SUPABASE_ANON_KEY`

### Шаг 2. Фото Диляры
1. Создай папку `public/photos/`.
2. Положи туда фото, например `hero.jpg`.
3. В `public/index.html` найди строку `const HERO_PHOTO = "";`
   и впиши путь: `const HERO_PHOTO = "/photos/hero.jpg";`
4. (Опционально) Можно добавить фото к конкретному вопросу — допиши в объект
   вопроса поле `photo`, например:
   `{ q:"Угадай Диляру в детстве", a:[...], correct:1, photo:"/photos/kid.jpg" }`

### Шаг 3. Деплой (Vercel, бесплатно)
1. Зарегистрируйся на https://vercel.com (можно через GitHub).
2. Установи CLI и задеплой из папки проекта:
   ```
   npm i -g vercel
   vercel
   ```
3. Когда спросит настройки — соглашайся со значениями по умолчанию.
4. Добавь переменные окружения (в дашборде Vercel → Project → Settings →
   Environment Variables, либо командой):
   ```
   vercel env add SUPABASE_URL
   vercel env add SUPABASE_ANON_KEY
   ```
   Вставь значения из Шага 1.
5. Передеплой в продакшн:
   ```
   vercel --prod
   ```
6. Готово! Vercel выдаст ссылку вида `https://dilyara-quiz.vercel.app` —
   её и рассылай гостям.

## ✅ Как проверить, что всё работает
- Открой ссылку, введи имя, пройди викторину.
- В конце должна появиться таблица лидеров и статистика (прошли / средний балл / рекорд).
- Если вместо таблицы видишь «Таблица лидеров недоступна» — значит не
  подхватились переменные `SUPABASE_URL` / `SUPABASE_ANON_KEY` (проверь Шаг 1 и 3.4).

## ✏️ Что ещё легко поменять (в public/index.html)
- Текст вопросов и варианты — массив `QUESTIONS` в начале скрипта.
- Уточни число стран (вопрос 11) — сейчас правильный ответ «45».
- Цвета — в блоке `:root` (зелёный `--green`, золотой `--gold`).
